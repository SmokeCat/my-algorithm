* Array | 数组

* LinkedList (single / double) | 链表 （单 / 双）
* Tree / Binary Tree | 树 / 二叉树
* Binary Search Tree | 二叉搜索树

* Greedy | 贪婪算法

* Breadth-first Search | 广度优先
* Depth-first Search | 深度优先
* Divide and Conquer | 分治
* Dynamic Programming | 动态规划
* Binary Search | 二分查找
* Graph | 图
